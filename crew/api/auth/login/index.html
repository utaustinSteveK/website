    POST /auth/login¶

    Authenticate a user and obtain access tokens for API access.

    Endpoint¶

    POST /api/v1/auth/login

    Description¶

    This endpoint authenticates a user with username and password credentials, returning JWT access and refresh tokens on successful authentication. The access token can be used to authenticate subsequent API requests.

    Authentication¶

    Public endpoint - No authentication required.

    Rate Limiting¶

    • 5 attempts per 15 minutes per IP address
    • Rate limit headers are included in the response

    Request¶

    Content Type¶

    Content-Type: application/json

    Request Body¶

    Field Type Required Description
    username string Yes Username for authentication (3-50 characters)
    password string Yes User password (minimum 8 characters)

    JSON Schema¶

    {
  "type": "object",
  "required": ["username", "password"],
  "properties": {
    "username": {
      "type": "string",
      "minLength": 3,
      "maxLength": 50,
      "description": "Username for authentication"
    },
    "password": {
      "type": "string",
      "minLength": 8,
      "description": "User password"
    }
  }
}

    Example Request¶

    {
  "username": "john.doe",
  "password": "SecurePass123!"
}

    Response¶

    Success Response (200 OK)¶

    Field Type Description
    access_token string JWT access token for API authentication
    refresh_token string JWT refresh token for renewing access
    token_type string Token type (always "Bearer")
    expires_in integer Access token expiration time in seconds
    user object Basic user information

    User Object¶

    Field Type Description
    id string User UUID
    username string Username
    email string Email address
    first_name string First name
    last_name string Last name
    role string User role
    is_active boolean Account status

    Examples¶

    cURL¶

    curl -X POST http://localhost:8000/api/v1/auth/login \
     -H "Content-Type: application/json" \
     -d '{
       "username": "john.doe",
       "password": "SecurePass123!"
     }'

    # Store response in variable
response=$(curl -s -X POST http://localhost:8000/api/v1/auth/login \
                -H "Content-Type: application/json" \
                -d '{
                  "username": "john.doe",
                  "password": "SecurePass123!"
                }' \
                -w "%{http_code}")

# Extract status code
http_code="${response: -3}"
response_body="${response%???}"

if [ "$http_code" -eq 200 ]; then
    echo "Login successful"
    # Extract token
    token=$(echo "$response_body" | jq -r '.access_token')
    echo "Access token: $token"
else
    echo "Login failed with status: $http_code"
    echo "$response_body"
fi

    import requests
from requests.exceptions import RequestException

def login_user(username: str, password: str, base_url: str = "http://localhost:8000"):
    """
    Login user and return authentication data.

    Returns:
        dict: Authentication data on success
        None: On failure
    """
    try:
        url = f"{base_url}/api/v1/auth/login"
        payload = {
            "username": username,
            "password": password
        }

        response = requests.post(url, json=payload, timeout=30)

        if response.status_code == 200:
            return response.json()
        elif response.status_code == 401:
            error_data = response.json()
            print(f"Authentication failed: {error_data.get('error', {}).get('message')}")
        elif response.status_code == 429:
            print("Rate limit exceeded. Please try again later.")
        elif response.status_code == 422:
            error_data = response.json()
            field_errors = error_data.get('error', {}).get('details', {}).get('field_errors', {})
            for field, errors in field_errors.items():
                print(f"{field}: {', '.join(errors)}")
        else:
            print(f"Unexpected error: {response.status_code}")

    except RequestException as e:
        print(f"Network error during login: {e}")

    return None

# Usage
auth_data = login_user("john.doe", "SecurePass123!")
if auth_data:
    print("Login successful!")
    access_token = auth_data["access_token"]

    JavaScript/TypeScript¶

    async function loginUser(username, password) {
    try {
        const response = await fetch('http://localhost:8000/api/v1/auth/login', {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
            },
            body: JSON.stringify({
                username: username,
                password: password
            })
        });

        const data = await response.json();

        if (response.ok) {
            // Store tokens securely (not in localStorage for production!)
            sessionStorage.setItem('access_token', data.access_token);
            sessionStorage.setItem('refresh_token', data.refresh_token);

            console.log('Login successful:', data.user);
            return data;
        } else {
            console.error('Login failed:', data.error.message);
            return null;
        }
    } catch (error) {
        console.error('Network error:', error);
        return null;
    }
}

// Usage
loginUser('john.doe', 'SecurePass123!')
    .then(authData => {
        if (authData) {
            console.log('Authenticated as:', authData.user.username);
        }
    });

    import axios from 'axios';

const api = axios.create({
    baseURL: 'http://localhost:8000/api/v1',
    headers: {
        'Content-Type': 'application/json'
    }
});

async function login(username, password) {
    try {
        const response = await api.post('/auth/login', {
            username,
            password
        });

        const { access_token, refresh_token, user } = response.data;

        // Set default authorization header
        api.defaults.headers.common['Authorization'] = `Bearer ${access_token}`;

        return {
            accessToken: access_token,
            refreshToken: refresh_token,
            user: user
        };
    } catch (error) {
        if (error.response) {
            // Server responded with error status
            const { status, data } = error.response;
            console.error(`Login failed (${status}):`, data.error.message);
        } else {
            // Network error
            console.error('Network error:', error.message);
        }
        throw error;
    }
}

// Usage
try {
    const authData = await login('john.doe', 'SecurePass123!');
    console.log('Login successful:', authData.user);
} catch (error) {
    console.error('Authentication failed');
}

    Notes¶

    Security Considerations¶

    1. Always use HTTPS in production to protect credentials in transit
    2. Store tokens securely - avoid localStorage for web applications
    3. Handle token expiration - implement automatic refresh logic
    4. Monitor failed attempts - implement account lockout policies

    Token Usage¶

    1. Access Token: Use for all authenticated API requests
    2. Refresh Token: Use only to obtain new access tokens
    3. Token Expiration: Access tokens expire in 30 minutes, plan for renewal
    4. Token Storage: Store securely and clear on logout

    Best Practices¶

    1. Validate input on the client side before sending requests
    2. Handle all error cases gracefully
    3. Implement retry logic with exponential backoff for rate limits
    4. Log authentication events for security monitoring
