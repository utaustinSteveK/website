    Vessels API¶

    The Vessels API provides comprehensive endpoints for fleet management, including vessel information, crew assignments, compliance tracking, and operational status management.

    Overview¶

    The Vessel Management system handles all aspects of fleet operations:

    • Fleet Information: Vessel specifications, technical details, and documentation
    • Crew Assignments: Managing crew deployments and rotations
    • Compliance Tracking: Certificates, inspections, and regulatory requirements
    • Operational Status: Vessel availability, location, and operational state
    • Port Management: Port calls, berth assignments, and logistics
    • Maintenance Scheduling: Dry dock, surveys, and maintenance planning

    Vessel Categories¶

    Vessel Types¶

    The system supports various types of maritime vessels:

    Type Description Crew Size Range
    container Container cargo vessels 20-25
    bulk_carrier Dry bulk cargo carriers 20-23
    tanker Oil/chemical tankers 22-28
    general_cargo General cargo vessels 18-22
    reefer Refrigerated cargo vessels 20-24
    roro Roll-on/Roll-off ferries 15-30
    cruise Passenger cruise ships 800-2000
    offshore Offshore support vessels 12-25

    Vessel Status¶

    Status Description
    active In service and operational
    dry_dock Undergoing maintenance or repairs
    laid_up Temporarily out of service
    under_construction Being built or converted
    sold Sold or transferred out of fleet
    scrapped Decommissioned and scrapped

    Data Model¶

    Vessel Object¶

    Field Type Description
    id UUID Unique vessel identifier
    name string Vessel name
    imo_number string International Maritime Organization number
    call_sign string Radio call sign
    mmsi string Maritime Mobile Service Identity
    vessel_type string Type of vessel
    flag_state string Country of registration
    port_of_registry string Official port of registration
    year_built integer Year of construction
    gross_tonnage number Gross tonnage (GT)
    net_tonnage number Net tonnage (NT)
    deadweight number Deadweight tonnage (DWT)
    length_overall number Length overall (LOA) in meters
    beam number Beam (width) in meters
    draft number Maximum draft in meters
    crew_capacity integer Maximum crew capacity
    status string Current operational status

    Vessel Assignment¶

    Field Type Description
    id UUID Assignment identifier
    vessel_id UUID Vessel being assigned to
    crew_member_id UUID Crew member assigned
    rank string Rank/position on vessel
    start_date date Assignment start date
    end_date date Assignment end date (if known)
    sign_on_port string Port where crew member joins
    sign_off_port string Port where crew member leaves
    is_active boolean Whether assignment is currently active

    Endpoints¶

    Core Vessel Operations¶

    • GET /vessels - List vessels with filtering
    • POST /vessels - Create new vessel
    • GET /vessels/{id} - Get vessel details
    • PUT /vessels/{id} - Update vessel information
    • DELETE /vessels/{id} - Remove vessel from fleet

    Crew Assignment Operations¶

    • GET /vessels/{id}/assignments - List current crew assignments
    • POST /vessels/{id}/assignments - Create crew assignment
    • PUT /vessels/{id}/assignments/{assignment_id} - Update assignment
    • DELETE /vessels/{id}/assignments/{assignment_id} - End assignment

    Compliance Operations¶

    • GET /vessels/{id}/certificates - List vessel certificates
    • POST /vessels/{id}/certificates - Add certificate
    • GET /vessels/{id}/compliance-status - Get compliance status
    • GET /vessels/{id}/inspections - List inspections

    Operational Operations¶

    • GET /vessels/{id}/position - Get current position
    • POST /vessels/{id}/position - Update position
    • GET /vessels/{id}/voyage - Get current voyage info
    • POST /vessels/{id}/port-call - Record port call

    Reporting Operations¶

    • GET /vessels/reports/fleet-status - Fleet status overview
    • GET /vessels/reports/crew-deployment - Crew deployment report
    • GET /vessels/reports/compliance - Fleet compliance report
    • GET /vessels/reports/utilization - Vessel utilization report

    Fleet Management Workflow¶

    import requests
from datetime import datetime, date, timedelta

class VesselAPI:
    def __init__(self, base_url: str, access_token: str):
        self.base_url = base_url
        self.headers = {
            "Authorization": f"Bearer {access_token}",
            "Content-Type": "application/json"
        }

    def list_active_vessels(self):
        """Get all active vessels in the fleet"""
        response = requests.get(
            f"{self.base_url}/api/v1/vessels",
            params={"is_active": True},
            headers=self.headers
        )
        return response.json() if response.status_code == 200 else None

    def get_vessel_crew(self, vessel_id: str):
        """Get current crew assignments for a vessel"""
        response = requests.get(
            f"{self.base_url}/api/v1/vessels/{vessel_id}/assignments",
            params={"is_active": True},
            headers=self.headers
        )
        return response.json() if response.status_code == 200 else None

    def assign_crew_member(self, vessel_id: str, crew_member_id: str, assignment_data: dict):
        """Assign crew member to vessel"""
        data = {
            "crew_member_id": crew_member_id,
            "vessel_id": vessel_id,
            **assignment_data
        }
        response = requests.post(
            f"{self.base_url}/api/v1/vessels/{vessel_id}/assignments",
            json=data,
            headers=self.headers
        )
        return response.json() if response.status_code == 201 else None

    def check_compliance_status(self, vessel_id: str):
        """Check vessel compliance status"""
        response = requests.get(
            f"{self.base_url}/api/v1/vessels/{vessel_id}/compliance-status",
            headers=self.headers
        )
        return response.json() if response.status_code == 200 else None

    def get_vessels_with_expiring_certificates(self, days: int = 30):
        """Find vessels with certificates expiring soon"""
        vessels = self.list_active_vessels()
        if not vessels:
            return []

        expiring_vessels = []
        for vessel in vessels['items']:
            compliance = self.check_compliance_status(vessel['id'])
            if compliance and compliance.get('certificates_expiring_soon', 0) > 0:
                expiring_vessels.append({
                    'vessel': vessel,
                    'compliance': compliance
                })

        return expiring_vessels

# Usage
vessel_api = VesselAPI("http://localhost:8000", access_token)

# Get fleet overview
active_vessels = vessel_api.list_active_vessels()
if active_vessels:
    print(f"Active vessels in fleet: {active_vessels['total']}")

    for vessel in active_vessels['items']:
        print(f"- {vessel['name']} ({vessel['vessel_type']}) - {vessel['flag_state']}")

        # Check crew status
        crew = vessel_api.get_vessel_crew(vessel['id'])
        if crew:
            print(f"  Current crew: {len(crew['items'])}/{vessel['crew_capacity']}")

            for assignment in crew['items']:
                print(f"    {assignment['crew_member_name']} - {assignment['rank']}")

# Check compliance across fleet
compliance_issues = vessel_api.get_vessels_with_expiring_certificates(60)
if compliance_issues:
    print("\nVessels with expiring certificates:")
    for item in compliance_issues:
        vessel = item['vessel']
        compliance = item['compliance']
        print(f"- {vessel['name']}: {compliance['certificates_expiring_soon']} certificates expiring")

# Assign crew member to vessel
assignment_data = {
    "rank": "Chief Officer",
    "start_date": str(date.today() + timedelta(days=7)),
    "sign_on_port": "Singapore",
    "contract_duration_months": 4
}

assignment = vessel_api.assign_crew_member(
    vessel_id="vessel-uuid-here",
    crew_member_id="crew-member-uuid-here",
    assignment_data=assignment_data
)

if assignment:
    print(f"Assigned crew member to {assignment['vessel_name']}")

    class FleetManager {
    constructor(baseUrl, accessToken) {
        this.baseUrl = baseUrl;
        this.headers = {
            'Authorization': `Bearer ${accessToken}`,
            'Content-Type': 'application/json'
        };
    }

    async getFleetOverview() {
        const response = await fetch(`${this.baseUrl}/api/v1/vessels`, {
            headers: this.headers
        });
        return response.ok ? await response.json() : null;
    }

    async getVesselDetails(vesselId) {
        const response = await fetch(
            `${this.baseUrl}/api/v1/vessels/${vesselId}`,
            { headers: this.headers }
        );
        return response.ok ? await response.json() : null;
    }

    async assignCrewToVessel(vesselId, assignments) {
        const results = [];

        for (const assignment of assignments) {
            const response = await fetch(
                `${this.baseUrl}/api/v1/vessels/${vesselId}/assignments`,
                {
                    method: 'POST',
                    headers: this.headers,
                    body: JSON.stringify(assignment)
                }
            );

            if (response.ok) {
                results.push(await response.json());
            } else {
                console.error(`Failed to assign ${assignment.crew_member_id}`);
            }
        }

        return results;
    }

    async monitorFleetCompliance() {
        const vessels = await this.getFleetOverview();
        if (!vessels) return;

        const complianceReport = [];

        for (const vessel of vessels.items) {
            const compliance = await this.getVesselCompliance(vessel.id);
            if (compliance) {
                complianceReport.push({
                    vessel: vessel.name,
                    imo: vessel.imo_number,
                    status: compliance.overall_status,
                    expiring_soon: compliance.certificates_expiring_soon || 0,
                    expired: compliance.expired_certificates || 0
                });
            }
        }

        return complianceReport;
    }

    async getVesselCompliance(vesselId) {
        const response = await fetch(
            `${this.baseUrl}/api/v1/vessels/${vesselId}/compliance-status`,
            { headers: this.headers }
        );
        return response.ok ? await response.json() : null;
    }

    async planCrewRotation(vesselId, rotationPlan) {
        // End current assignments
        const currentCrew = await this.getVesselCrew(vesselId);

        for (const crewMember of currentCrew.items) {
            if (rotationPlan.off_signers.includes(crewMember.crew_member_id)) {
                await this.endAssignment(vesselId, crewMember.id, rotationPlan.sign_off_date);
            }
        }

        // Create new assignments
        const newAssignments = rotationPlan.on_signers.map(signer => ({
            crew_member_id: signer.crew_member_id,
            rank: signer.rank,
            start_date: rotationPlan.sign_on_date,
            sign_on_port: rotationPlan.port,
            contract_duration_months: signer.contract_duration
        }));

        return await this.assignCrewToVessel(vesselId, newAssignments);
    }

    async getVesselCrew(vesselId) {
        const response = await fetch(
            `${this.baseUrl}/api/v1/vessels/${vesselId}/assignments?is_active=true`,
            { headers: this.headers }
        );
        return response.ok ? await response.json() : null;
    }

    async endAssignment(vesselId, assignmentId, endDate) {
        const response = await fetch(
            `${this.baseUrl}/api/v1/vessels/${vesselId}/assignments/${assignmentId}`,
            {
                method: 'PUT',
                headers: this.headers,
                body: JSON.stringify({
                    end_date: endDate,
                    is_active: false
                })
            }
        );
        return response.ok;
    }
}

// Usage
const fleetManager = new FleetManager('http://localhost:8000', accessToken);

// Monitor fleet compliance
async function checkFleetCompliance() {
    const report = await fleetManager.monitorFleetCompliance();

    console.log('Fleet Compliance Report:');
    report.forEach(vessel => {
        console.log(`${vessel.vessel} (${vessel.imo}):`);
        console.log(`  Status: ${vessel.status}`);
        console.log(`  Expiring: ${vessel.expiring_soon} certificates`);
        console.log(`  Expired: ${vessel.expired} certificates`);
    });

    // Alert for critical issues
    const criticalVessels = report.filter(v =>
        v.expired > 0 || v.status === 'non_compliant'
    );

    if (criticalVessels.length > 0) {
        console.log('\n⚠️ CRITICAL COMPLIANCE ISSUES:');
        criticalVessels.forEach(vessel => {
            console.log(`- ${vessel.vessel}: ${vessel.expired} expired certificates`);
        });
    }
}

// Plan crew rotation
async function planRotation(vesselId) {
    const rotationPlan = {
        port: 'Singapore',
        sign_off_date: '2024-02-15',
        sign_on_date: '2024-02-16',
        off_signers: ['crew-uuid-1', 'crew-uuid-2'],
        on_signers: [
            {
                crew_member_id: 'crew-uuid-3',
                rank: 'Master',
                contract_duration: 4
            },
            {
                crew_member_id: 'crew-uuid-4',
                rank: 'Chief Officer',
                contract_duration: 4
            }
        ]
    };

    const results = await fleetManager.planCrewRotation(vesselId, rotationPlan);
    console.log(`Planned rotation: ${results.length} new assignments created`);
}

// Run compliance check
checkFleetCompliance();

    Best Practices¶

    Fleet Management¶

    1. Maintain accurate vessel data - Keep technical specifications up to date
    2. Monitor compliance proactively - Track certificate expiration dates
    3. Optimize crew deployment - Balance experience levels and contract terms
    4. Plan maintenance windows - Coordinate dry dock and survey scheduling
    5. Track performance metrics - Monitor fuel consumption, speed, and efficiency

    Operational Efficiency¶

    1. Real-time tracking - Maintain current vessel position and status
    2. Voyage optimization - Plan efficient routes and port calls
    3. Crew rotation planning - Minimize gaps in crew coverage
    4. Emergency response - Maintain rapid crew replacement capabilities
    5. Cost management - Track and analyze operational expenses

    Regulatory Compliance¶

    1. Certificate management - Ensure all documents are valid and current
    2. Inspection scheduling - Plan surveys and inspections in advance
    3. Flag state requirements - Comply with registration country regulations
    4. Port state control - Prepare for random inspections
